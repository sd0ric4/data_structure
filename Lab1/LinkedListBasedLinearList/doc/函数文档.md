## InitList

- **作用**: 初始化一个空的链表。
- **算法实现**:
  1. 分配一个头结点的内存空间。
  2. 将头结点的 `next` 指针域置为 `NULL`，表示链表为空。
- **参数**:

  | 参数名 | 类型        | 描述           |
  | ------ | ----------- | -------------- |
  | `L`    | `LinkList*` | 指向链表的指针 |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

## ClearList

- **作用**: 清空链表中的所有元素。
- **算法实现**:
  1. 遍历链表，释放每个节点的内存空间。
  2. 将头结点的 `next` 指针域置为 `NULL`。
- **参数**:

  | 参数名 | 类型       | 描述         |
  | ------ | ---------- | ------------ |
  | `L`    | `LinkList` | 链表的头指针 |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

## ListLength

- **作用**: 获取链表的长度。
- **算法实现**:
  1. 遍历链表，计数节点个数。
- **参数**:

  | 参数名 | 类型       | 描述         |
  | ------ | ---------- | ------------ |
  | `L`    | `LinkList` | 链表的头指针 |

- **返回**:
  - `int`：链表中数据元素的个数。

## GetElem

- **作用**: 获取链表中第 `i` 个数据元素的值。
- **算法实现**:
  1. 遍历链表，找到第 `i` 个节点。
  2. 返回该节点的数据域的值。
- **参数**:

  | 参数名 | 类型        | 描述                     |
  | ------ | ----------- | ------------------------ |
  | `L`    | `LinkList`  | 链表的头指针             |
  | `i`    | `int`       | 元素的位置               |
  | `e`    | `ElemType*` | 用于存储返回的元素的指针 |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

## LocateElem

- **作用**: 查找链表中第一个与 `e` 相等的数据元素的指针。
- **算法实现**:
  1. 遍历链表，找到第一个与 `e` 相等的节点。
- **参数**:

  | 参数名 | 类型       | 描述           |
  | ------ | ---------- | -------------- |
  | `L`    | `LinkList` | 链表的头指针   |
  | `e`    | `ElemType` | 要查找的元素值 |

- **返回**:
  - `LNode*`：指向找到的节点的指针，若未找到则返回 `NULL`。

## ListInsert

- **作用**: 在链表的第 `i` 个位置插入一个数据元素 `e`。
- **算法实现**:
  1. 遍历链表，找到第 `i-1` 个节点。
  2. 分配一个新节点，将数据域置为 `e`。
  3. 将新节点插入链表。
- **参数**:

  | 参数名 | 类型        | 描述           |
  | ------ | ----------- | -------------- |
  | `L`    | `LinkList*` | 指向链表的指针 |
  | `i`    | `int`       | 插入的位置     |
  | `e`    | `ElemType`  | 要插入的元素值 |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

## ListDelete

- **作用**: 删除链表的第 `i` 个数据元素，并用 `e` 返回其值。
- **算法实现**:
  1. 遍历链表，找到第 `i-1` 个节点。
  2. 删除第 `i` 个节点，并返回其数据域的值。
- **参数**:

  | 参数名 | 类型        | 描述                     |
  | ------ | ----------- | ------------------------ |
  | `L`    | `LinkList*` | 指向链表的指针           |
  | `i`    | `int`       | 删除的位置               |
  | `e`    | `ElemType*` | 用于存储删除的元素的指针 |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

  - 无

## Union

- **作用**: 将两个链表合并，结果存储在第一个链表中。
- **算法实现**:
  1. 遍历第二个链表，将不在第一个链表中的元素插入第一个链表。
- **参数**:

  | 参数名 | 类型        | 描述                 |
  | ------ | ----------- | -------------------- |
  | `La`   | `LinkList*` | 指向第一个链表的指针 |
  | `Lb`   | `LinkList`  | 第二个链表的头指针   |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

- **参数**:

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

## MergeList

- **作用**: 将两个有序链表合并为一个新的有序链表。
- **算法实现**:
  1. 初始化一个新的链表。
  2. 依次比较两个链表的元素，将较小的元素插入新链表。
  3. 将剩余的元素插入新链表。
- **参数**:

  | 参数名 | 类型        | 描述                   |
  | ------ | ----------- | ---------------------- |
  | `La`   | `LinkList`  | 第一个有序链表的头指针 |
  | `Lb`   | `LinkList`  | 第二个有序链表的头指针 |
  | `Lc`   | `LinkList*` | 指向新链表的指针       |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。

## Purge

- **作用**: 去掉链表中的重复元素。
- **算法实现**:
  1. 遍历链表，对于每个元素，删除其后续节点中与其相等的元素。
- **参数**:

  | 参数名 | 类型        | 描述           |
  | ------ | ----------- | -------------- |
  | `Lc`   | `LinkList*` | 指向链表的指针 |

- **返回**:
  - `Status`：函数执行结果，成功返回 `OK`，失败返回错误码。
