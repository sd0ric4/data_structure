## 函数介绍

### InitHTree

- **作用**: 初始化赫夫曼树。
- **算法实现**:
  1. 计算赫夫曼树的总节点数。
  2. 初始化前 n 个节点为叶子节点，设置权重和父子关系。
  3. 初始化剩余节点为非叶子节点，设置权重和父子关系。
- **参数**:

  | 参数名 | 类型   | 描述       |
  | ------ | ------ | ---------- |
  | HT     | HTree& | 赫夫曼树   |
  | w      | int\*  | 权重数组   |
  | n      | int    | 叶子节点数 |

- **返回**:
  - `void`

---

### Select

- **作用**: 从赫夫曼树中选择两个最小权重且父节点为空的结点。
- **算法实现**:
  1. 初始化 s1 和 s2 为-1。
  2. 遍历所有节点，找到两个最小权重且父节点为空的节点。
- **参数**:

  | 参数名 | 类型  | 描述             |
  | ------ | ----- | ---------------- |
  | HT     | HTree | 赫夫曼树         |
  | n      | int   | 当前节点数       |
  | s1     | int&  | 最小权重节点索引 |
  | s2     | int&  | 次小权重节点索引 |

- **返回**:
  - `void`

---

### CreateHTree

- **作用**: 构造赫夫曼树。
- **算法实现**:
  1. 计算赫夫曼树的总节点数。
  2. 依次选择两个最小权重的节点，构造新的父节点。
  3. 更新父子关系和权重。
- **参数**:

  | 参数名 | 类型   | 描述       |
  | ------ | ------ | ---------- |
  | HT     | HTree& | 赫夫曼树   |
  | n      | int    | 叶子节点数 |

- **返回**:
  - `void`

---

### HTCoding

- **作用**: 生成赫夫曼编码。
- **算法实现**:
  1. 初始化编码数组和临时存放编码的数组。
  2. 对每个叶子节点，逆向追踪其父节点，生成编码。
  3. 将生成的编码存入编码数组。
  4. 释放临时存放编码的数组。
- **参数**:

  | 参数名 | 类型    | 描述       |
  | ------ | ------- | ---------- |
  | HT     | HTree   | 赫夫曼树   |
  | HC     | HTCode& | 赫夫曼编码 |
  | n      | int     | 叶子节点数 |

- **返回**:
  - `void`
